Limit
=====

Resource limitation.

WallTime limit
--------------

Execution time limit.

.. code-block:: none

    'limit': {
        'slug': 'limit:walltime',
        'value': <hours>:<minutes>:<seconds>
    }

Iteration limit
---------------

Limit on the number of iterations.

.. code-block:: none

    'limit': {
        'slug': 'limit:iteration',
        'value': <number>
    }
